LegBot

The LEGO Discord bot

The end of the ! command...

Since August 31 2022, Discord decided to remove the ability for the bots to read the content of messages in chat. This is a very good initiative to protect privacy, as a bot could basically be a message collector on every server without anyone knowing.

Unfortunately that means that Legbot can not read the part of your message with an exclamation point and answer to that.

I tried to apply for the "intent" to have access to the messages, but Discord decided that the features were not worth it.

Please enjoy Legbot as before using / commands now!

All commands

/set [set number] Show a set data
/search [query] Search for a set by name (Beta feature, may be surprisingly wrong)
/part [part number] Show a part data (Bricklink ID)
/part " search terms " Search for a part name (Beta, not usabe in the middle of a sentence)
/setoftheday Get Brickset's random set of the day
/review [set number] Get the set score based on the BrickInsights review
/legbot Show all the bot commands
/credits Display the bot credits
