LegBot

The LEGO Discord bot

Features

Get a LEGO set
Returns informations about a certain set.
!set 8723
Get a LEGO part
Returns available data about a certain LEGO piece.
No more cross purposes when debating about your moc!
  • Piece still in production ?
  • Similar molds
  • Shopping links
!part 15712
... and more!
How bad is this set ?
!review 75098
Get a direct link to Brickset about the set
!bs 8723
Get a direct link to Bricklink about the set
!bl 8723
Display the LegBot commands
!help
Display the LegBot credits
!credits

Credits

LegBot is using the following datasources and APIs :
  • Rebrickable API
  • Brick Insight public API
  • Brickset API
  • BrickLink links
  • BrickOwl links
LegBot is based on the DiscordJS library.
